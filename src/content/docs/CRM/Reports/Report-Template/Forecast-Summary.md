Utilize the **Forecast Summary** report to generate a list report containing the following opportunity forecast information:  
* **Company name**  
* **Opportunity Description** 
* **Representative**  
* **Status** 
* **Priority**
* **% to Close** 
* **% to Buy** 
* **Expected Close** 
* **$ Amount** 
* With a Summarized **Total Amount** at the bottom
