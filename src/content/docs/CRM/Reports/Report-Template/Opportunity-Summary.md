Utilize the **Opportunity Summary** report to generate a list report containing the following opportunity information:  
* **Company name**  
* **Opportunity description**  
* **Status** 
* **Priority** 
* **% to Close** 
* **% to Buy** 
* **Expected Close** 
* **$ Amount**
