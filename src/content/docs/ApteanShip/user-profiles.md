The Profile menu item under the username displays the user details and rights for the currently logged-in user. Aptean Ship uses Roles to define and limit access to functionality, and these Roles are assigned to the Users you create.

![](assets/images/user-profile-image1.png)

### User Details

Users details can be managed by an Administrator in the Subscription Management System under **Users & Roles**. The Profile displays the Role(s) assigned to the User and other personal information.

### User Rights

In the Profile, you can click **Edit** to edit that user's details and rights in the SMS.
