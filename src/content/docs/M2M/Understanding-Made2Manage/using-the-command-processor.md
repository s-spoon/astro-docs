draft: false
type: Article

---

Use the **M2M Processor Service Configuration (PROC)** page to:

-	Start or stop the M2M Processor Service.
-	Check if the Processor Service is running.
-	Enable or disable the Processor Services.
-	View currently executing processes.
-	Configure companies and intervals for each processor service.
-	Select the days when the processor service runs.
-	Manually execute a process.
